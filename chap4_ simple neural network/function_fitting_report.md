## 2. 函数定义

结合当前 notebook 中的可视化切片

```python
test_x = np.array([[4, 3, 2, x] for x in x_dim3])
```

以及真值曲线形状，可以将当前实验对应的目标函数写为：

\[
f(x_1,x_2,x_3,x_4)=x_1^2+x_2^2+x_3^2+x_4^2
\]

当固定前三维为 `(4, 3, 2)` 时，有：

\[
f(4,3,2,x)=4^2+3^2+2^2+x^2=29+x^2
\]

这与图中的蓝色真值曲线一致。
